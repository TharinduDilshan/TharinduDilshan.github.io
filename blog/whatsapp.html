Have a problem with WhatsApp New Update???


How to find out which version of Android you have

  • Samsung Galaxy Phones: “About Phone” > “Software Information”
  • Stock Android: “System” -> “About Phone” or “About Tablet”

If your android version is KitKat or (< Android 4.0.3), there is an issue with WhatsApp new update(Version: 2.18.232 - beta). So don't update Whatsapp (Version: 2.18.232 - beta). If you have already installed it and if it is not working, you can download a previous version and reuse the WhatsApp.
